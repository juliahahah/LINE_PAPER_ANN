# PAPER CALL 群組指令

## 初始化

| 指令 | 用途 |
|---|---|
| `初始化管理員` | 第一位管理員執行一次 |
| `加入我輪值` | 自己加入輪值隊尾 |
| `加入輪值` + 標註成員 | 管理員加入指定成員 |
| `輪值名單` | 查看目前輪值順序 |

## 會議設定

| 指令 | 範例 |
|---|---|
| 設定固定週會 | `設定週會 星期二 10:00` |
| 設定下次單次會議 | `設定下次會議 2026/07/16 10:00` |
| 查看下次輪值 | `輪值` |

## 報告處理

| 指令 | 用途 |
|---|---|
| `請假` | 本次報告人請假，立即補下一位；本人下次優先補報告 |
| `幫請假` + 標註成員 | 管理員替一位本次報告人請假 |
| `完成` | 本次報告人完成，移到輪值隊尾 |
| 按「本週不開會」 | 管理員跳過本週；輪值順序保持不變 |

## 輪值規則

- 每次預設安排前兩位。
- 成員完成後才會移到隊尾，所以每個人會依序輪到一次。
- 請假者不會被略過，會在下一個實際開會週優先補報告。
- 替補者完成後也會移至隊尾。

## 每日提醒

Apps Script 觸發條件要選 `sendDailyReminder`。

- 設定為每分鐘：僅測試用途。
- 設定為日計時器：正式使用。
- 程式每次執行都會檢查，但只有「明天有會議」時才發送提醒。
- 同一場會議只會提醒一次，不會每分鐘重複洗群組。
